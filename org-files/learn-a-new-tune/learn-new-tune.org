** Embellish the melody

Here's the first melodic cell of Have You Met Miss Jones

#+begin_src  lilypond :exports results :file melody_raw.pdf
  
  #(set-global-staff-size 22)

  \paper {
  #(set-paper-size "letter")
  tagline = ##f
  ragged-last-bottom = ##t
  ragged-bottom = ##t
  }

  \score {
    \new StaffGroup <<
      \new ChordNames { \chordmode { f1:maj7 } } 
      \new Staff {
        \global
        \key c \major
        \relative c'' {
          a4. d8 c d4 c8~
          c1
        }
      }
    >>
  }
#+end_src
